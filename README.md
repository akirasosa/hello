# Hello Worktrees

A minimal Python "Hello World" application demonstrating Git worktrees functionality.

## Prerequisites

- Python 3.x

## Installation

```bash
git clone https://github.com/akirasosa/hello-worktrees.git
cd hello-worktrees
```

## Usage

Run the application:

```bash
python hello.py
```

Expected output:

```
Hello World
```

## Project Structure

```
hello-worktrees/
└── hello.py    # Main application file
```
