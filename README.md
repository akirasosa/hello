# Hello

A minimal Python "Hello World" application.

## Description

This is a simple Python script that prints "Hello World" to the console.

## Requirements

- Python 3.x

## Installation

Clone the repository:

```bash
git clone ssh://git@github.com/akirasosa/hello.git
cd hello
```

## Usage

Run the script:

```bash
python hello.py
```

Output:
```
Hello World
```
